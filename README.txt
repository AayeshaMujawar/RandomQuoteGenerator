# Random Quote Generator

Ye ek simple Python project hai jo ek CSV file se random quotes pick karke show karta hai.

## Kaise Run Kare
1. Is project ko download karo
2. Folder open karo
3. quotes.csv file me apne quotes add karo
4. Python file run karo:
   python quote_generator.py

## Example Output
"Life is what happens when you're busy making other plans." - John Lennon
